1. Concepts de Virtualisation

  • Type 1 (Bare Metal) : Installé directement sur le matériel (ESXi, Proxmox). Meilleures perfs.
  • Type 2 (Hosted) : Installé sur un OS (VirtualBox, VMware Workstation). Pour le test/dev.

2. Proxmox VE

Solution Open Source basée sur Debian.

Différence Clé : Proxmox gère les VM (KVM) (isolation totale) et les Conteneurs (LXC) (partage du noyau, plus léger).

3. Stockage & Réseau Virtuel

  • Stockage : LVM-Thin (Provisionnement dynamique), ZFS, Ceph (Distribué).
  • Réseau : Linux Bridge (vmbr0) pour relier les VM à la carte physique.

4. Stratégie de Sauvegarde

Ne pas confondre Snapshot et Backup.

  • Snapshot : "Photo" à un instant T. Utile avant une mise à jour. N'est PAS une sauvegarde (si le disque meurt, le snapshot aussi).
  • Backup : Copie complète de la VM exportée sur un autre support (NAS, Cloud).
