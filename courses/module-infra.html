1. Concepts & Hyperviseurs

Comprendre la différence fondamentale entre les types d'hyperviseurs :

  • Type 1 (Bare Metal) : Installé directement sur le matériel (ESXi, Proxmox, Hyper-V Server).
  • Type 2 (Hosted) : Installé sur un OS existant (VirtualBox, VMware Workstation).

Examen : Savoir justifier le choix d'un Type 1 en production pour les performances et la sécurité.

2. Proxmox & VMware

Gestion des machines virtuelles (VM) et des conteneurs (LXC).

  • Proxmox : Basé sur Debian, utilise KVM pour les VM et LXC pour les conteneurs.
  • VMware vSphere : Solution propriétaire leader, gestion via vCenter.

3. Stockage & Réseaux Virtuels

Comment les VM communiquent et stockent leurs données.

  • Virtual Switching : Bridge (Pont), NAT, et isolation LAN.
  • Stockage : Provisionnement mince (Thin) vs Épais (Thick), snapshots et sauvegardes.

4. Architecture Hybride

L'interconnexion entre les mondes Windows et Linux (NTiers).

Exemple : Bridge Réseau Proxmox (/etc/network/interfaces)
auto vmbr0 iface vmbr0 inet static address 192.168.1.10/24 gateway 192.168.1.1 bridge-ports eth0
