1. DNS (Le pilier de l'AD)

Le DNS permet la résolution des noms en adresses IP. Sans lui, les clients ne peuvent pas localiser les contrôleurs de domaine.

Info Examen : Un serveur DNS Windows utilise des "Zones intégrées à l'AD" pour sécuriser et répliquer les données.

PowerShell - Vérification DNS
Resolve-DnsName -Name dc01.formation.local

2. ADDS & Rôles FSMO

L'Active Directory centralise la gestion des utilisateurs. Il s'appuie sur 5 rôles uniques (FSMO).

Les 5 Rôles : Schéma, Nommage (Forêt) + PDC, RID, Infrastructure (Domaine).

3. Messagerie & Exchange

Comprendre les flux de messagerie est essentiel.

  • SMTP (Port 25) : Transfert de serveur à serveur.
  • IMAP/POP : Réception client.

4. GPO (Stratégies de groupe)

L'application des GPO suit l'ordre LSDOU : Local, Site, Domaine, OU.
