1. Modèle OSI & Encapsulation

Fondamental pour le diagnostic (Troubleshooting). Une panne se situe toujours sur une couche précise.

Astuce Examen : "Layer 8" désigne l'utilisateur (l'humain) qui est souvent la cause du problème.

2. Switching L2 (VLAN, Trunk, STP)

La segmentation est la base de la sécurité réseau.

  • VLAN (802.1Q) : Isole les domaines de diffusion.
  • Trunk : Transporte plusieurs VLANs entre switchs.
  • STP (Spanning Tree) : Empêche les boucles de commutation (Broadcast Storm).
Cisco IOS - Config Trunk & Native VLAN
Switch(config)# int gi0/1 Switch(config-if)# switchport mode trunk Switch(config-if)# switchport trunk native vlan 99 Switch(config-if)# switchport trunk allowed vlan 10,20,99

3. Routage L3 (Interconnexion)

Le routeur prend ses décisions via la table de routage (Adresse IP Destination).

  • Router-on-a-Stick : Routage inter-VLAN via une sous-interface.
  • OSPF : Protocole dynamique à état de liens (Link-State). Utilise l'algo Dijkstra.

4. Pare-feu & VPN (pfSense)

Sécurisation de la bordure (Edge) du réseau.
